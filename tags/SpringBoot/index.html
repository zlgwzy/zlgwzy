SpringBoot中注册bean的方法

spring在启动的时候自动将bean注册到spring容器，实现控制反转

注册bean的三种方式

  1. @Import
  2. @Bean
  3. @ComponentScan

1.@Import

//@Import注解
public @interface Import {
Class<?>[] value();
}
接收类的class对象数组，spring容器可以根据class对象的newInstane()方法实例化类，或者getDeclaredConstructor()方法获取构造方法实例化类

在启动类使用@Import(value = {TestBean.class} ),就可以在项目启动的时候将bean注册到spring容器，这是最直接的方式将指定的类注册到ioc容器

SpringBoot实现拦截器

1.什么是拦截器？

拦截器(Interceptor)同过滤器(Filter)一样，都是面向切面编程——AOP思想的具体实现。

拦截器可以在Controller处理请求之前编写日志，添加或更新配置。

在 Spring中，当请求发送到Controller时，在被Controller处理之前，它必须经过Interceptors（0或更多）。
