5.常用命令

  1. systemctl start vsftpd.service //启动ftp服务
  2. systemctl enable vsftpd.service //设置开机自启动
  3. netstat -antup | grep ftp //查看ftp服务端口
  4. systemctl restart vsftpd.service //重启ftp服务
  5. systemctl status vsftpd.service //查看ftp服务状态
