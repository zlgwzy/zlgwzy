从零开始部署一个Java项目(基于centos7环境)

1.项目部署需要的准备工作

  1. 一台服务器(云服务器或物理服务器)
  2. 安装jdk
  3. 安装tomcat
  4. 安装mysql
  5. 安装nginx

2.安装JDK

  1. 查看可以安装的java环境

  2. 安装jkd

  3. 查看jdk版本

2.安装tomcat

  1. tomcat官网下载安装包

  2. 将下载好的tomcat放在/local/src目录下并解压

  3. 开放端口

  4. 启动tomcat

    启动tomcat的三种方式 * 直接启动 ./startup.sh

    • 作为服务启动 nohup ./startup.sh &

      初次使用的情况下会提示权限不足

      chmod +x *.sh 使用该命令给.sh提供权限

    • 控制台动态输出方式启动 ./catalina.sh run 动态地显示tomcat后台的控制台输出信息,Ctrl+C后退出并关闭服务

      参考文章：linux下启动tomcat服务的命令是什么

  1. 查看tomcat启动效果

3.安装myslq(5.6)

  1. 下载yum源(无法使用wget，yum install wget 安装wget)

  2. 安装mysql yum源(rpm文件下载到执行wget命令的目录下,el7对应centos7)

  3. 安装mysql

  4. 设置开机自启(2,3,4都为开)

  5. 启动mysql

  6. 设置密码

  7. 建立远程root用户

  8. 开放3306端口

  9. 修改字符编码为utf8


4.安装nginx

参考文章：Nginx初级教程(一)Nginx概述与安装

5.前端项目打包dist文件夹下的内容移动到后端项目的static目录下，打包项目为可执行jar

6.服务器运行可执行jar

图片上传的demo可正常运行



7.服务器运行war

SpringMVC项目和SpringBoot项目打包成war有不同区别
参考另一篇文章：SpringMVC和SpringBoot打包成war的注意事项
